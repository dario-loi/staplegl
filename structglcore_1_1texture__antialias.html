Detailed Description

OpenGL texture details relating to the number of samples to use for the texture.

Warning
Antialiasing is mainly intended for use in framebuffer attachments, so if you are creating a texture to be used as an object's material, you most likely want to use glcore::tex_samples::MSAA_X1.
